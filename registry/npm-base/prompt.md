Ensure Node.js (version 18 or newer) and npm are available. Check with
`node --version` and `npm --version`.

If npm is missing, install Node with the platform's package manager:

- Debian / Ubuntu:  apt-get install -y nodejs npm
- Fedora:           dnf install -y nodejs
- macOS:            brew install node
- Windows:          winget install OpenJS.NodeJS.LTS

Do not proceed until `npm --version` prints a version.
